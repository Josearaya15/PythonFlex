Proyecto de una web de consultoría en donde se crean formularios para inscribir una empresa, enviar una petición de diagnostico rápido y para añadir un proyectos. Luego se puede hacer petición para buscar un proyecto en particular, editarlo o eliminarlo.
Además, cuenta con login de usuario junto con edición del perfil y cambio de contraseña.
